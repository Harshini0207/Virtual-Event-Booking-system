# Virtual Event Booking Management System

A web-based application developed to simplify virtual event registration and booking. This system allows users to browse events, book tickets, and manage reservations, 
while admins can create and manage events efficiently.

## Features
- User Signup & Login  
- Browse and Book Events  
- Cancel Bookings  
- Admin Dashboard  
- Event Management  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  

## Purpose
To provide an easy and efficient platform for managing virtual event bookings online.
